# Space Weather May 2024 – Slovenia Event Dataset
This repository contains datasets and methodological documentation for the study of the May 2024 space weather event and its cascading impacts on environmental and critical infrastructure systems in Slovenia.

## Repository structure
1_sources  
Original reports and source materials used for event extraction.
List of sources.

2_tso  
Transmission system operator (ELES) reports and extracted events.

3_events  
Event-level datasets:
- raw_events
- raw_ops_events
- daily_stats
- daily_context

4_daily_integrated  
Integrated daily dataset combining environmental forcing and sector impacts.

5_metrics  
Definitions and methodology of the CSCI metrics.

6_csci_outputs  
Computed CSCI daily index results.

dataset  
Final datasets used in the paper.

figures  
Figures used in the publication.

methodology  
Detailed methodological documentation.
